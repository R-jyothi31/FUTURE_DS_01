# FUTURE_DS_01 - Business Sales Performance Analytics

## Project Overview

This project focuses on analyzing business sales data to identify revenue trends, top-performing products, category performance, and regional sales patterns. The goal is to transform raw sales data into meaningful business insights that support strategic decision-making.

Using Python for data analysis and visualization, the project evaluates key business metrics such as total sales, profit, order volume, and customer purchasing patterns.

## Objectives

* Analyze overall business sales performance
* Identify top-selling products and categories
* Evaluate regional and state-wise sales trends
* Track monthly sales and profit growth
* Generate actionable business recommendations
* Build visualizations to support decision-making

## Dataset

**Dataset Used:** Sample Superstore Dataset

The dataset includes:

* Orders
* Products
* Categories
* Sales
* Profit
* Regions
* States
* Customer Information
